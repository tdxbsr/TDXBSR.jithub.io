# 刘鸿天的个人博客

这是一个简单的个人博客网站，使用HTML和CSS构建。

## 网站结构

```
├── index.html          # 首页
├── about.html          # 关于我
├── blog.html           # 博客文章列表
├── contact.html        # 联系方式
├── style.css           # 样式文件
├── posts/              # 博客文章目录
│   ├── first-week.html            # 大学第一周的感受
│   └── programming-journey.html   # 我的编程之旅开始了
└── README.md           # 说明文档
```

## 如何使用

1. 克隆或下载本项目到本地
2. 使用浏览器打开 `index.html` 文件即可查看网站
3. 可以通过修改HTML文件来更新网站内容

## 如何添加新的博客文章

1. 在 `posts/` 目录下创建一个新的HTML文件
2. 复制现有文章的模板结构，修改标题、内容和日期
3. 在 `blog.html` 中添加新文章的链接和摘要
4. 在 `index.html` 的最新文章部分添加新文章的卡片

## 自定义样式

可以通过修改 `style.css` 文件来自定义网站的样式，包括颜色、字体、布局等。

## 技术栈

- HTML5
- CSS3
- 响应式设计

## 浏览器兼容性

支持现代浏览器，包括 Chrome、Firefox、Safari、Edge 等。
